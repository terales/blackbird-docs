XTRF is een uitgebreid vertaalbeheersysteem dat vertaalworkflows stroomlijnt en automatiseert voor taaldienstverleners en bedrijfsvertaalafdelingen. Het biedt robuuste functies voor projectbeheer, samenwerking en automatisering om de efficiëntie van vertaalprocessen te verbeteren.

## Acties

### Classic - Job 

- **Get job** Informatie over een job ophalen
- **Update job** Een job bijwerken, waarbij alleen de velden worden gespecificeerd die bijgewerkt moeten worden
- **Upload output file for job** Een outputbestand uploaden voor een job

### Classic - Project 

- **Create language combination for project** Een nieuwe taalcombinatie maken voor een klassiek project zonder een taak aan te maken
- **Create new project** Een nieuw klassiek project aanmaken

- **List custom fields for project** De aangepaste velden voor een klassiek project weergeven, waarbij velden worden geretourneerd die aan het project kunnen worden toegewezen zonder hun waarden. Om de veldwaarde te verkrijgen, gebruik de actie "Get custom field for project" die overeenkomt met het veldtype

- **Update project** Een klassiek project bijwerken, waarbij alleen de velden worden gespecificeerd die bijgewerkt moeten worden
